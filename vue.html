Content Edit System --- This is the content edit system designed by Vue.js. The purpose of starting this project is to quickly build the website through editing web content on the admin page. Designing a specific database structure to identity different section of this website with multiple contents. The admin page is to do CRUD(create, read, update, delete) job to manage web content. Once there is any change in the backend system, it will render to the landing page immediately.
